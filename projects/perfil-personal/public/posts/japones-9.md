# Aprendiendo japonés con un gordo barbudo #9

## Gramática básica con Cure Dolly

### Lección 7

Hoy vamos a empezar a negar con 「ない」; este adjetivo es el más básico y, como tal, sirve para negar cualquier cosa. Dolly empieza a sarandear sobre el hecho de que es un adjetivo y no sé qué; acá no nos interesa demasiado, porque en español hacemos más o menos lo mismo.

#### 「ない」

Este adjetivo 「い」 niega cosas y significa "no es" o "no tiene".

##### Negación de verbos

- Ichidan

Como el resto de estos verbos, la negación es simple:

```estructura
「る」 → 「ない」
```

```text
たべる → たべない
```

El significado de la negación es simple: no se hace el verbo.

- Godan

En el caso de los godan, la negación es un poco más compleja: tenemos que cambiar el verbo a la rama 「あ」 y le sumamos 「ない」 (ahora vamos a hablar de eso).

```text
のむ → のまない

いく → いかない

まつ → またない
```

- Irregulares

Acá el verbo es irregular y negarlo es cambiarlo:

```text
する → しない

くる → こない
```

##### 「ない」 es un adjetivo

Como cualquier adjetivo, puede usarse en diferentes tiempos.

| Forma           | Ejemplo (たべない)                      |
| --------------- | --------------------------------------- |
| Presente        | たべない                                |
| Pasado          | たべなかった                            |
| Negativo pasado | たべなくなかった (muy raro)             |
| Adverbial       | たべなく (para conectar con なる, etc.) |

##### 「ない」 de existencia

Negar la existencia de un objeto (que no haya un objeto) es simple: 「ない」.

```text
ある → ない
```

```text
おかね が ない
じかん が ない
```

```traduccion
No tengo dinero.
No hay tiempo.
```

#### Rama 「あ」

En japonés, los verbos godan cambian su última vocal según lo que queramos hacer con ellos.

Estas vocales se llaman ramas (bases) del verbo. Para negar un verbo godan, debemos cambiarlo a la rama 「あ」 y agregar 「ない」. Esto varía dependiendo de la terminación de la raíz del verbo.

| Terminación diccionario | Rama-あ | Rama-い | Rama-う | Rama-え | Rama-お |
| ----------------------- | ------- | ------- | ------- | ------- | ------- |
| う                      | わ      | い      | う      | え      | お      |
| く                      | か      | き      | く      | け      | こ      |
| す                      | さ      | し      | す      | せ      | そ      |
| つ                      | た      | ち      | つ      | て      | と      |
| ぬ                      | な      | に      | ぬ      | ね      | の      |
| む                      | ま      | み      | む      | め      | も      |
| ぶ                      | ば      | び      | ぶ      | べ      | ぼ      |
| る (godan)              | ら      | り      | る      | れ      | ろ      |

##### Ejemplos

| Verbo | Rama「あ」 | Negativo |
| ----- | ---------- | -------- |
| 買う  | 買わ       | 買わない |
| 書く  | 書か       | 書かない |
| 泳ぐ  | 泳が       | 泳がない |
| 話す  | 話さ       | 話さない |
| 待つ  | 待た       | 待たない |
| 死ぬ  | 死な       | 死なない |
| 遊ぶ  | 遊ば       | 遊ばない |
| 読む  | 読ま       | 読まない |
| 走る  | 走ら       | 走らない |

###### Notas

- La rama 「う」 pasa a 「わ」 en lugar de 「あ」. Por lo que dicen, es para no tener doble 「あ」 y que sea más fácil de pronunciar.
- Todo es hiperregular, a excepción de los verbos irregulares (que son dos).
