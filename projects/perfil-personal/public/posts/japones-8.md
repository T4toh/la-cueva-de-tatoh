## Gramática básica con Cure Dolly

### Lección 6

En esta lección nuestra maestra digital empieza a hablar de los adjetivos para terminra con el 「の」, yo voy a hacer al revés, más que nada porque la partícula es fácil de entender y ya la usamos en esta misma lección.

#### Partícula 「の」

La partícula 「の」 une dos cosas y marca una relación de pertenencia o especificación.
En español suele equivaler a “de” o funciona como un adjetivo posesivo (“mi/tu/su”), según el caso. En inglés es fácil de ver que equivale a "'s" (Tatoh's blog/ el blog de Tatoh). En español tenemos que invertir el uso (comparado con el inglés.)

El 「の」 es una partícula simple con algunos usos.

- Pertenencia

Este es el caso más simple y común (por lo que me han dicho)

```text
わたし の ほん。
```

```traduccion
Mi libro.
```

```estructura
[A] 「の」 [B]。
```

- Descripción

En este caso el 「の」 se usa para describir algo (cosa que vamos usar en esta misma lección)

```text
にほん の くるま。
```

```traduccion
Auto japonés.
```

- Material

En este caso el 「の」 se usa para decir que algo es de un material.

```text
きん の ゆびわ。
```

```traduccion
Anillo de oro.
```

- Sustantivador

Hace que una frase o adjetivo pueda usarse de sustantivo. Esto lo vamos a ver un poco más adelante.

```text
あかい の。
```

```traduccion
El rojo.
```

#### Adjetivos

En ponja hay tres tipos de adjetivos (más bien palabras que describien).

- Adjetivos 「い」（い形容詞）
- Adjetivos 「な」（な形容詞）
- Sustantivos + だ (Por lo que leí en el genki y otras fuentes **Cure Dolly**, a veces los juntan con los 「な」, pero técnicamente son otra cosa)

A diferencia del español, los adjetivos japoneses tienen función verbal: algunos conjugan tiempo, otros se comportan como sustantivos con cópula.

##### Adjetivos 「い」（い形容詞）

Estos son verbales. Funcionan como verbos porque cambian de forma según el tiempo.

- 「たかい」 (alto / caro)
- 「さむい」 (frío)
- 「おいしい」 (rico)

| Forma           | Ejemplo        |
| --------------- | -------------- |
| Presente        | たかい         |
| Negativa        | たかくない     |
| Pasado          | たかかった     |
| Pasado negativo | たかくなかった |

Funcionan como verbos, sin だ ni cópula.

##### Adjetivos 「な」（な形容詞）

No tienen forma verbal. Funcionan como sustantivos + だ, y para describir un nombre llevan el な en medio.

- しずか (tranquilo)
- ゆうめい (famoso)
- きれい (lindo / limpio)

```text
しずかな へや。
```

```traduccion
Habitación tranquila.
```

```estructura
「しずか」 「な」 [Sustantivo]
```

```text
へや は しずかだ。
```

```traduccion
La habitación es tranquila.
```

```estructura
 [Sustantivo] 「は」 「しずか+だ」
```

Para pasado, negación, etc., se comportan como la cópula:

```estructura
しずかだ → しずかじゃない → しずかだった → しずかじゃなかった
```

##### Sustantivos + だ

Cure Dolly los separa de los な, porque realmente no son adjetivos, sino sustantivos que describen.

Ejemplos:

- ほんとう (verdad)
- あんぜん (seguridad)
- ばか (idiota)

```text
ほんとうの はなし。
```

```traduccion
Historia verdadera.
```

```estructura
「ほんとう+の」 [Sustantivo]
```

```text
はなし は ほんとうだ。
```

```traduccion
La historia es verdad.
```

```estructura
[Sustantivo] 「は」 「ほんとう+だ」
```

No llevan な cuando predican:

```estructura
X es Y → Y es un sustantivo y usa だ
```

##### Verbos + 「の」 para describir sustantivos

En japonés, cuando querés usar un verbo para describir un sustantivo, podés usar la estructura:

```estructura
[Verbo en forma diccionario] + 「の」 + [Sustantivo]
```

Acá 「の」 funciona como un conector que vuelve al verbo “descriptivo”, parecido a decir “(algo) que VERBO”.

```text
たべる の ひと。
```

```traduccion
Persona que come.
```

```text
はしる の こども。
```

```traduccion
El niño que corre.
```

```text
ねる の ねこ。
```

```traduccion
El gato que duerme.
```

###### Notas

- El verbo describe a un sustantivo.
- 「の」 convierte esa frase verbal en una especie de “adjetivo”.
- No es un adjetivo real, pero cumple el mismo rol: modificar un sustantivo.

| Tipo                   | Se usa como              | Conjugación    | Ejemplo             |
| ---------------------- | ------------------------ | -------------- | ------------------- |
| **Adjetivos い**       | verbos                   | sí             | たかい → たかかった |
| **Adjetivos な**       | sustantivos + な         | con だ         | しずか → しずかだ   |
| **Sustantivos + だ**   | sustantivos descriptivos | con だ         | ほんとうだ          |
| **Verbo + sustantivo** | verbo descriptivo        | según el verbo | あるいている ひと   |
