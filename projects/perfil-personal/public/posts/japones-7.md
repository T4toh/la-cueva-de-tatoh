# Aprendiendo japonés con un gordo barbudo #7

## Gramática básica con Cure Dolly

### Lección 5.5

En esta segunda parte vamos a ver cómo armar la forma 「て」. Lo primero que hay que decir, y que Cure Dolly repite todo el tiempo, es que en ponja no se conjugan los verbos; los verbos cambian para usarse con verbos auxiliares (ya sé que suena igual y que overlapean una bocha, pero confunde muchísimo cuando no lo hacen). Hay muchos verbos auxiliares y, a medida que pasen las lecciones, vamos a ir aprendiéndolos (「ている」, 「たい」, 「た」, etc.).

#### 「一段」

Estos son los fáciles; casi todos terminan en 「る」 y para usarlos solo cambiás 「る」 por lo que necesitás, como ya vimos en la parte anterior.

#### 「五段動詞」

| Terminación diccionario | Cambia a | Ejemplo           | Resultado               |
| ----------------------- | -------- | ----------------- | ----------------------- |
| う / つ / る            | って     | あう → あって     | encontrar → encontrando |
| む / ぶ / ぬ            | んで     | よむ → よんで     | leer → leyendo          |
| く                      | いて     | あるく → あるいて | caminar → caminando     |
| ぐ                      | いで     | およぐ → およいで | nadar → nadando         |
| す                      | して     | はなす → はなして | hablar → hablando       |

La teoría, por lo que vengo leyendo, es sobre los sonidos que decimos y todo es muy natural, cosa que no lo es para mí, pero 100% seguro que es por la falta de práctica y de inmersión. Así que hay dos maneras: mirás la tabla hasta que te acordás o te ponés a leer hasta que lo interiorizás, como todo buen estudiante.

#### 「不規則動詞」

Los dos que vimos en la media clase anterior están aquí.

| Verbo | Forma て | Nota                                     |
| ----- | -------- | ---------------------------------------- |
| する  | して     | Se comporta como los que terminan en す. |
| くる  | きて     | Irregular total                          |

#### Notas

La forma 「て」 no indica tiempo. Es solo un conector o marcador de aspecto (acción en curso, acciones conectadas, etc.). Sirve para unir oraciones (“y luego…”) o para usar con auxiliares, como 「〜ている」 (presente continuo), 「〜てください」 (pedir), 「〜てもいい」 (permiso), etc.

Técnicamente, 「だ」 (ser) también tiene una forma equivalente: 「で」, y todo lo que aprendimos acá se usa de la misma manera (según Cure Dolly).
